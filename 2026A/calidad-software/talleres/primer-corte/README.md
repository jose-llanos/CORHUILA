### Introducción
La Calidad de Software se enfoca en garantizar que un sistema no solo funcione, sino que lo haga de manera correcta, segura y eficiente.

En esta materia aprendemos a evaluar, mejorar y asegurar la calidad en cada etapa del desarrollo, aplicando buenas prácticas, pruebas y estándares.
