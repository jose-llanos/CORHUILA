## 📌 Descripción

Este proyecto tiene como objetivo evaluar la calidad del software mediante la aplicación de la métrica Defect Density utilizando la herramienta SonarQube. Se realizó un análisis sobre el código fuente del proyecto Java "defect-density", identificando un total de 22 issues clasificados en niveles de severidad alta, media y baja.

A partir de estos resultados, se calcularon los defectos por cada mil líneas de código (KLOC), permitiendo determinar el nivel de calidad del sistema. Posteriormente, se implementaron mejoras basadas en buenas prácticas de programación, incluyendo refactorización, optimización de código, validaciones y aplicación de principios de código limpio.

Finalmente, se ejecutó nuevamente el análisis en SonarQube, evidenciando una reducción significativa de defectos y una mejora en la calidad del software.
