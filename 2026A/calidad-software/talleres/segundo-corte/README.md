## 📝 Descripción

Este taller tiene como objetivo evaluar la calidad del código fuente mediante el uso de la herramienta SonarQube, identificando problemas como bugs, vulnerabilidades y malas prácticas de programación. A partir del análisis inicial, se calcularon métricas como la densidad de defectos (Defect Density), lo cual permitió determinar el nivel de calidad del software.

Posteriormente, se implementaron mejoras en el código aplicando principios de código limpio, buenas prácticas y técnicas de refactorización. Esto incluyó la eliminación de variables innecesarias, la reducción de la complejidad cognitiva, el uso adecuado de comparaciones y el manejo correcto de excepciones.

Finalmente, se realizó un segundo análisis en SonarQube, evidenciando una mejora significativa en las métricas de calidad, logrando reducir los issues a cero y alcanzando un código más mantenible, confiable y alineado con estándares de desarrollo profesional.
