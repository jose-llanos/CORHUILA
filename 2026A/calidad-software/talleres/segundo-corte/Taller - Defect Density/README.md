## 📋 Descripción
 
Este taller aborda el análisis y mejora de la calidad del código fuente del proyecto **defect-density**, desarrollado en Java, mediante el uso de **SonarQube** e **IntelliJ IDEA**. Se identificaron **22 issues** (1 high, 14 medium y 7 low), se calculó la **Defect Density** con base en las líneas de código del proyecto, y se implementaron soluciones para cada issue aplicando principios de **código limpio**, **encapsulamiento**, **manejo adecuado de excepciones**, **uso de loggers**, **reducción de complejidad cognitiva** y **eliminación de código muerto**. Finalmente, se compararon los resultados antes y después de las correcciones, logrando reducir los issues a **0** y mejorando significativamente la calidad general del proyecto.
