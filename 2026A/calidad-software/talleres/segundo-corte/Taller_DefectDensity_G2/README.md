## Introducción  

El presente proyecto expone los resultados de un ejercicio práctico orientado a la
aplicación de la métrica Defect Density, utilizando como stack tecnológico SonarQube, Docker,
Java e IntelliJ IDEA. Como objeto de estudio se seleccionó un proyecto académico con defectos
sembrados de manera intencional, lo cual facilitó la identificación sistemática de issues
asociados a errores de programación, vulnerabilidades de seguridad y desviaciones de buenas
prácticas de codificación
