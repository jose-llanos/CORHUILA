## Introducción

En este proyecto se implementa un sistema de conversión de temperaturas entre grados Celsius y Fahrenheit, el cual sirve como caso práctico para aplicar conceptos de calidad de software.
A través del uso de herramientas como JUnit y JaCoCo, se busca validar el correcto funcionamiento del sistema, asegurando precisión en los resultados, manejo de errores y cumplimiento de los requisitos establecidos.
