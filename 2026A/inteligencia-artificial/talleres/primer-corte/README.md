## 📌 Introducción

En este taller se aborda el estudio de la **Inteligencia Artificial (IA)** y las **Redes Neuronales Artificiales**, analizando cómo las máquinas pueden aprender a partir de datos tomando como referencia el funcionamiento de la neurona biológica.

Se desarrollan conceptos como el **Perceptrón**, sus limitaciones frente a problemas no lineales, y las **Redes Neuronales Multicapa (MLP)** junto con el algoritmo de **Backpropagation**.  

Además, se realizan implementaciones prácticas en Python para comprender cómo estos modelos pueden resolver problemas como el **XOR**.

---

## 🎯 Objetivo General

Analizar y comprender el funcionamiento del **Perceptrón simple** y de las **Redes Neuronales Multicapa**, implementándolos desde cero en Python para entender su comportamiento matemático y su capacidad para resolver problemas lineales y no lineales.

---

## 🎯 Objetivos Específicos

- Explicar los fundamentos de la Inteligencia Artificial  
- Describir el modelo matemático del Perceptrón  
- Analizar sus limitaciones frente a problemas no lineales (XOR)  
- Implementar una red neuronal multicapa con Backpropagation  
- Evaluar el desempeño del modelo en el problema XOR  

---

## 🧠 Marco Teórico

Se estudian los fundamentos de la Inteligencia Artificial como rama de la informática enfocada en el desarrollo de sistemas capaces de aprender y tomar decisiones.

Se analiza la relación entre la **neurona biológica** y la **neurona artificial**, donde:

- Entradas → señales recibidas  
- Pesos → importancia de cada entrada  
- Función de activación → decisión de salida  

Se profundiza en:

- 🔹 El **Perceptrón** como modelo lineal  
- 🔹 Sus limitaciones en problemas no lineales  
- 🔹 Las **MLP** como solución a estos problemas  
- 🔹 El algoritmo de **Backpropagation** para entrenamiento

## ⚙️ Desarrollo del Taller

### 🔵 5.1 Fundamentos de IA y Neurona Biológica
Se explican los conceptos básicos de IA, sus aplicaciones y la analogía entre la neurona biológica y el perceptrón.

---

### 🔵 5.2 El Perceptrón

- Modelo matemático  
- Función de activación  
- Algoritmo de entrenamiento  
- Limitaciones (problema XOR)  

Se demuestra que el perceptrón solo puede resolver problemas **linealmente separables**.

---

### 🔵 5.3 Redes Neuronales Multicapa (MLP)

- Estructura: entrada, capas ocultas, salida  
- Funciones de activación (Sigmoid, ReLU, Tanh)  
- Algoritmo de Backpropagation  
- Ventajas frente al perceptrón  

Se demuestra cómo las MLP pueden resolver problemas no lineales como el XOR.

## ✅ Conclusiones

El taller permitió comprender la evolución desde el perceptrón hasta las redes neuronales multicapa.

Se evidenció que:

- El perceptrón funciona para problemas lineales  
- No puede resolver problemas como XOR
