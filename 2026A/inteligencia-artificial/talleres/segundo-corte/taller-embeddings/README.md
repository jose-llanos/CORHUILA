# Taller: Representaciones de Texto - Word Embeddings y Embedding Contextual

## Descripcion

Taller practico donde se exploran los embeddings como representaciones vectoriales de palabras que capturan relaciones semanticas y sintacticas.

## Contenido

1. **Conceptos Fundamentales** - Comparacion entre One-Hot Encoding y Dense Embeddings
2. **Word2Vec** - Entrenamiento de un modelo Word2Vec con Skip-gram usando Gensim
3. **Visualizacion con t-SNE** - Reduccion de dimensionalidad para visualizar embeddings en 2D
4. **Clasificacion de Textos** - Uso de embeddings promedio con Logistic Regression para clasificar sentimientos

## Dependencias

- `gensim`
- `nltk`
- `scikit-learn`
- `matplotlib`
- `numpy`
